# Blog Pessoal - Testes com Jest

Projeto mínimo em NestJS com recurso de Usuário e teste E2E.

## Instalação

```bash
npm install
```

## Rodar projeto

```bash
npm run start:dev
```

## Rodar testes E2E

```bash
npm run test:e2e
```

## Rotas

- POST `/usuarios/cadastrar`
- POST `/usuarios/logar`
- GET `/usuarios/all`
- PUT `/usuarios/atualizar`
